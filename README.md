# Customer_Behavior_Analysis
🛍️ Customer Behavior Analysis

📌 Project Overview

This project analyzes customer shopping behavior using a dataset of 3,900 purchase transactions to uncover patterns in spending, customer demographics, product preferences, and subscription behavior.

The objective is to generate data-driven business insights that help organizations improve marketing strategies, product positioning, and customer retention.

📊 Dataset Information

Rows: 3,900
Columns: 18

Key Attributes

- Customer demographics: Age, Gender, Location

- Purchase information: Item Purchased, Category, Purchase Amount

- Shopping behavior: Discount Applied, Purchase Frequency, Review Rating

- Logistics: Shipping Type

- Subscription Status

🛠 Tools & Technologies

- Python (Pandas) – Data cleaning, preprocessing, and EDA

- PostgreSQL / SQL – Business analysis queries

- Power BI – Interactive dashboard and visualization

🔎 Data Analysis Process

1️⃣ Data Cleaning, Preparation & EDA (Python)

- Handled missing values in Review Rating

- Standardized column names for better readability

- Performed Exploratory Data Analysis (EDA) to understand distributions and trends

- Created age_group feature for customer segmentation

- Removed redundant columns

- Loaded the cleaned dataset into PostgreSQL for SQL analysis

2️⃣ Business Analysis (SQL)

- Key business queries performed:

- Revenue by Gender

- High-spending customers using discounts

- Top-rated products

- Shipping type comparison

- Subscriber vs Non-subscriber spending

- Discount-dependent products

- Customer segmentation (New, Returning, Loyal)

- Revenue contribution by Age Group

📈 Power BI Dashboard

An interactive Power BI dashboard was built to visualize:

- Revenue by product category

- Customer age group distribution

- Subscription vs non-subscription analysis

- Product performance

- Purchase trends

💡 Key Business Insights

- Loyal customers generate higher overall revenue

- Discounts significantly influence purchase behavior

- Certain product categories perform better with targeted promotions

- Subscription users tend to spend more on average

🚀 Business Recommendations

- Introduce customer loyalty programs

- Improve subscription benefits

- Optimize discount strategies

- Focus marketing on high-value customer segments
